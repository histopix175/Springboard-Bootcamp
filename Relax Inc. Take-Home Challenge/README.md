# Relax Inc Take Home Challenge

This repository contains an analysis of user data for Relax Inc. to identify which factors predict future user adoption. 

## Project Overview
The goal of this challenge was to identify top predictors of 'adopted users' of the Relax Inc platform.

## Repository Structure
The project is organized into the following directories:

*   **Main Folder**: Contains the final summary report.
    *   `Relax Inc Take Home Challenge Findings.pdf`: A breif report of the methodology, exploratory data analysis (EDA), and modeling results.
*   **Notebooks/**: Contains the technical implementation and analysis.
    *   `Relax_Inc_Challenge.ipynb`: The Jupyter Notebook containing data cleaning, feature engineering, and the Logistic Regression model.
*   **Data/**: Contains the raw datasets used for the analysis.
    *   `takehome_users.csv`: User demographic and account information.
    *   `takehome_user_engagement.csv`: Row-level login data for all users.
